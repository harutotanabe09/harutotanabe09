# Githubのおすすめ機能

### コードオーナー

PR作成時に、レビュアーを自動的に入力するようにする

### PR テンプレート
PR作成時に、テンプレートを埋めるようにする

### ブランチ保護
レビューが１人以上ない限り、masterブランチへのマージ不可

### リリースタグ
本番環境への反映時に、リリースタグを用いてリリースを行う

### 脆弱性診断
パッケージの脆弱性を診断するようにdependabotという機能を利用

### Slack通知
コミットメッセージ、マージ、Issue起票などを行うとSlackのチャンネルに通知するように設定

### Jiraとの課題連携

課題ツール側で対象のコードが閲覧できる
